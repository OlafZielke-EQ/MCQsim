# MCQSIM COMPILATION
_MCQsim_ is written in _C_ and parallelized with _MPI_. It further uses _CBLAS_ (provided via _GSL_ or otherwise). These libraries must be accessible during compilation/run-time. _MCQsim_ compilation/installation is done in the command window i.e., terminal using the provided Makefile. This file should work for MacOS, Linux, and HPC.
